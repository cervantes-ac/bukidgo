# Bukidnon Unified Theme & Design System\n\n## Overview\n\nThis document outlines the unified Bukidnon theme and design system implemented across all pages: **Explore**, **Food**, **Guides**, and **Events**.\n\n## Color Palette\n\n### Primary Colors\n- **Dark Green** (`#0A3D2F`) - Deep forest, primary background\n- **Forest Green** (`#1E4D2B`) - Mountain accent\n- **Mountain Green** (`#2D6A4F`) - Secondary accent\n- **Light Green** (`#4A9D6F`) - Interactive elements\n\n### Accent Colors\n- **Gold** (`#F5C400`) - Tribal/cultural accent\n- **Gold Light** (`#FDD835`) - Hover states\n\n### Earth Tones\n- **Earth Brown** (`#8B4513`) - Text and borders\n- **Brown Accent** (`#C4622D`) - Secondary buttons\n- **Dark Brown** (`#5D3A1A`) - Deep text\n\n### Neutral Palette\n- **Cream** (`#F5F5F0`) - Main background\n- **Light Cream** (`#F5F0E8`) - Secondary background\n- **Beige** (`#E8DCC8`) - Borders\n- **Dark Text** (`#1A1208`) - Primary text\n- **Medium Text** (`#2D2D2D`) - Secondary text\n- **Light Text** (`rgba(245, 245, 240, 0.65)`) - Tertiary text\n\n## Typography\n\n### Font Families\n- **Serif** (`Playfair Display`) - Headlines, titles\n- **Body** (`Barlow`) - Main content\n- **Condensed** (`Barlow Condensed`) - Tags, labels, buttons\n- **Mono** (`JetBrains Mono`) - Code, technical text\n\n### Font Sizes\n- **Hero Title**: `clamp(2.5rem, 6vw, 4rem)`\n- **Section Title**: `clamp(1.8rem, 4vw, 2.8rem)`\n- **Card Title**: `20px`\n- **Body Text**: `15px`\n- **Small Text**: `12px`\n- **Tiny Text**: `10px`\n\n## Components\n\n### Buttons\n\n#### Primary Button (`.bk-btn-primary`)\n- Background: Gradient from `#9C2A2A` to `#7A1F1F`\n- Color: Cream\n- Padding: `12px 32px`\n- Font: Condensed, 700, 14px\n- Letter Spacing: `0.15em`\n- Hover: Darker gradient, translateY(-2px), shadow\n\n#### Outline Button (`.bk-btn-outline`)\n- Background: Transparent\n- Color: `#9C2A2A`\n- Border: `2px solid #9C2A2A`\n- Padding: `12px 26px`\n- Hover: Filled background, cream text\n\n### Cards (`.bk-card`)\n- Transition: `transform 0.3s cubic-bezier(0.22,1,0.36,1), box-shadow 0.3s ease`\n- Hover: `translateY(-8px)`, shadow xl\n- Image hover: `scale(1.08)`\n\n### Badges\n\n#### Primary Badge (`.badge-primary`)\n- Background: Gradient from `#9C2A2A` to `#7A1F1F`\n- Color: Cream\n- Padding: `6px 12px`\n- Font Size: `10px`\n- Font Weight: 700\n\n#### Accent Badge (`.badge-accent`)\n- Background: Gold\n- Color: Dark Green\n- Padding: `6px 12px`\n\n### Tags (`.bk-tag`)\n- Display: Inline-flex\n- Font: Condensed, 700, 11px\n- Letter Spacing: `0.25em`\n- Text Transform: Uppercase\n- Color: `#9C2A2A`\n- Includes decorative line before text\n\n## Spacing Scale\n\n- **xs**: `4px`\n- **sm**: `8px`\n- **md**: `16px`\n- **lg**: `24px`\n- **xl**: `32px`\n- **xxl**: `48px`\n- **xxxl**: `64px`\n\n## Border Radius\n\n- **none**: `0px`\n- **sm**: `2px` (default for cards, buttons)\n- **md**: `4px`\n- **lg**: `8px`\n- **full**: `99px` (circles, pills)\n\n## Shadows\n\n- **sm**: `0 2px 8px rgba(10, 61, 47, 0.1)`\n- **md**: `0 8px 24px rgba(10, 61, 47, 0.15)`\n- **lg**: `0 12px 40px rgba(10, 61, 47, 0.25)`\n- **xl**: `0 28px 72px rgba(10, 61, 47, 0.35)`\n\n## Gradients\n\n- **Mountain Dark**: `linear-gradient(135deg, #0A3D2F 0%, #1E4D2B 50%, #0A3D2F 100%)`\n- **Mountain Warm**: `linear-gradient(135deg, #1E4D2B 0%, #2D6A4F 100%)`\n- **Gold Accent**: `linear-gradient(135deg, #F5C400 0%, #FDD835 100%)`\n- **Brown Accent**: `linear-gradient(135deg, #8B4513 0%, #C4622D 100%)`\n- **Cream**: `linear-gradient(135deg, #F5F5F0 0%, #F5E8D5 100%)`\n\n## Animations\n\n### Float Animation\n```css\n@keyframes floatY {\n  0%, 100% { transform: translateY(0) }\n  50% { transform: translateY(-12px) }\n}\n.float-anim { animation: Layouts\n\n### Hero Section\n- Background: Mountain Dark gradient\n- Padding: `80px 2rem 60px`\n- Text Align: Center\n- Contains: Location badge, title, subtitle, search bar, stats\n\n### Filter **Tablet**: `768px`\n- **Desktop**: `1024px`\n- **Wide**: `1280px`\n\n### Mobile Adjustments\n- Hero title: `3.2rem`\n- Grid: Single column\n- Padding: Reduced\n- Font sizes: Scaled down\n\n##
